# 猫人内衣抖音 - 短视频周度每日计划表

这是一个用于管理猫人内衣抖音短视频拍摄和素材的周度计划表项目。

## ✨ 功能特性

- **周度计划制定**: 详细规划每日的拍摄任务、内容、脚本、产品、模特、场景等。
- **模块化管理**: 将每日工作划分为核心模块，如拍摄前准备、内容策划、拍摄执行、后期制作、数据复盘等。
- **状态追踪**: 实时更新各任务项的完成状态（待开始、进行中、已完成）。
- **数据持久化**: 支持将周计划保存到后端数据库，并能随时加载已保存的计划。
- **CSV导出**: 可将当前填写的表格内容导出为CSV文件，方便离线查看和分享。
- **前后端分离**: 前端使用HTML、CSS、JavaScript实现交互界面，后端使用Node.js (Express) 和 SQLite数据库进行数据存储和API服务。

## 🛠️ 技术栈

- **前端**: HTML, CSS, JavaScript (Vanilla JS)
- **后端**: Node.js, Express.js
- **数据库**: SQLite
- **版本控制**: Git, GitHub

## 🚀 本地运行指南

### 1. 克隆仓库 (如果您是从GitHub获取)

```bash
git clone https://github.com/qunanchang/maoren-video-plan.git
cd maoren-video-plan
```

### 2. 后端服务启动

确保您已安装 Node.js 和 npm。

```bash
cd backend
npm install
npm start
```

后端服务默认运行在 `http://localhost:3000`。

### 3. 前端页面访问

直接在浏览器中打开项目根目录下的 `index.html` 文件。

或者，如果您希望通过Web服务器访问（例如，如果您配置了本地服务器指向项目根目录），请确保服务器已启动。
